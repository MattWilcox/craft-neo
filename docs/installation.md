# Installation

## Instructions

1. [Download the latest version](https://github.com/spicywebau/craft-neo/archive/1.x.zip) of Neo and unzip.
2. Copy the `neo/` directory into your `craft/plugins/` directory
3. Inside the Craft control panel, go to *Settings &rarr; Plugins*
4. Find the row for Neo and click *Install*


## Requirements

### Craft version
Neo requires Craft CMS 2.6 or greater.

### PHP version
Neo requires PHP 5.4 or greater.

### Browser support
- IE10+
- Edge
- Chrome 40+
- Firefox 35+
- Opera 27+
- Safari 9+

While Neo may work on browsers and versions outside of those listed, the above list is what Neo explicitly supports. The best option is to use an up-to-date major browser.
